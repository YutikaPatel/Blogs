# Publish This Blog on GitHub Pages

1. Push this repo to GitHub.
2. Open GitHub repo `Settings` -> `Pages`.
3. Under `Build and deployment`:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`
4. Save.
5. Wait 1-2 minutes and open the Pages URL GitHub shows.

Your home page is `docs/index.html` and your flower article is in `docs/posts/flowers-first-post.html`.
